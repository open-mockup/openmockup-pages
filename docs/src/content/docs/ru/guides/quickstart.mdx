Самый быстрый способ пощупать OpenMockup — открыть его в браузере.
Без установки, без настройки проекта.

## 1. Откройте плейграунд

Перейдите в [Live Editor](/playground/) (временный маршрут на время
миграции доков — скоро будет подключён к существующему плейграунду).
Увидите сплит: разметка слева, рендер мокапа справа.

## 2. Попробуйте минимальный экран

Вставьте в редактор:

```jsx
<Card>
  <Heading>С возвращением</Heading>
  <Paragraph>Войдите, чтобы продолжить работу.</Paragraph>
  <Form>
    <Field label="Email" />
    <Field label="Пароль" type="password" />
    <Button variant="primary">Войти</Button>
  </Form>
</Card>
```

Появится карточка с заголовком, формой из двух полей и кнопкой.
Это и есть полный цикл: пишете декларативную разметку — получаете мокап.

## 3. Добавьте второй экран и свяжите их

Переключитесь в **Flow Editor**. Добавьте второй экран — например,
дашборд после входа. На первом экране у кнопки **Войти** укажите
`action`, который ведёт на этот второй экран. Прокликайте флоу в
превью, чтобы пройти по прототипу.
